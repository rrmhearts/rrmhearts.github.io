--- title: "Radar Visualizer" excerpt: "An interactive, React-based educational tool for visualizing radar antenna patterns." collection: portfolio --- An interactive, React-based educational tool for visualizing and understanding radar antenna radiation patterns, beam steering, and array physics. This simulator generates normalized 2D polar plots of various antenna types, allowing users to manipulate physical parameters like aperture size, element count (N), and element spacing (d), while observing the resulting changes to the main lobe, sidelobes, and grating lobes in real-time. The radar visualizer is built using React for the user interface and a canvas element for rendering the polar plots. It provides an intuitive interface for users to explore how different configurations affect the radiation pattern, making it an effective educational tool for students and professionals in the field of radar engineering and antenna design. The code is hosted on GitHub: Radar Visualization.
