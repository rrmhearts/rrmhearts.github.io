--- title: "Radar Visualizer" excerpt: "An interactive, React-based educational tool for visualizing radar antenna patterns.
" collection: portfolio --- An interactive, React-based educational tool for visualizing and understanding radar antenna radiation patterns, beam steering, and array physics. This simulator generates normalized 2D polar plots of various antenna types, allowing users to manipulate physical parameters like aperture size, element count (N), and element spacing (d), while observing the resulting changes to the main lobe, sidelobes, and grating lobes in real-time.
